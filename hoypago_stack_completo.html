Roadmap MVP
Datos clave
Frontend — App móvil
React Native + Expo recomendado
Un codebase, iOS y Android. Expo simplifica build y OTA updates. Comunidad enorme.
Flutter
Alternativa sólida si ya sabés Dart. Performance nativo superior pero curva de aprendizaje mayor.
Backend — API
Node.js + NestJS recomendado
TypeScript nativo, arquitectura modular, ideal para crecer. Soporte a REST + WebSockets.
Node.js + Express
Más simple para MVP rápido. Menos boilerplate, migrar a NestJS después si escala.
Base de datos
PostgreSQL principal
Usuarios, tarjetas, historial. En Supabase para setup en minutos.
Redis
Cache de promos y tasas. Evita scraping repetido. TTL de 1–6hs.
Supabase
Postgres gestionado + Auth + Realtime. Gratis hasta 500MB.
Motor de datos — el corazón de HoyPago
Scraper de promos bancarias
Puppeteer o Playwright. Corre cada 6hs. Fuentes: Galicia, BBVA, Santander, Macro, Naranja X, MODO, Mercado Pago. Resultados cacheados en Redis.
APIs de datos financieros
dolarapi.com · ambito.com/api · BCRA API para tasas oficiales. Inflación desde INDEC. Todas gratuitas.
Notificaciones y IA
Push Notifications
Expo Push API (gratis) o Firebase Cloud Messaging. Alertas diarias personalizadas según tarjetas del usuario.
IA ligera de recomendación
Claude API o GPT-4o mini para el motor de recomendación. Input: perfil de tarjetas + promos del día. Output: "tu mejor opción hoy".
Infraestructura y DevOps
Railway / Render
Backend Node.js. Deploy con push a main. Gratis para empezar.
Vercel
Landing page y dashboard web. CDN global, gratis.
GitHub Actions
CI/CD. Tests + lint + build automático en cada PR.

Modelo de negocio
Sponsored alerts Principal
Bancos pagan para que su promo se notifique destacada. Galicia, BBVA, Naranja X ya tienen budgets de marketing digital.
Afiliación fintechs Rápido
Ualá, Brubank, Mercado Pago pagan $5–15 USD por usuario nuevo referido. 100 referidos/mes = $500–1.500 USD.
Premium users
$3–5 USD/mes. Alertas avanzadas, simulador de cuotas, modo inversión, historial completo.
Data insights
Datos anónimos agregados: "Los martes 60% paga con QR en CABA". Venta a bancos y consultoras.
